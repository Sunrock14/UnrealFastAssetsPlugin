# Fast Assets

**Streamline your asset workflow with Fast Assets - the ultimate external asset browser for Unreal Engine.**

Fast Assets is a powerful editor plugin that allows you to browse, preview, and import assets from any external folder directly into your Unreal Engine project. Say goodbye to tedious file navigation and manual imports!

## Key Features

### External Folder Browsing
- Browse any folder on your computer directly within Unreal Engine
- Navigate through directories with a familiar interface
- Remember recent paths for quick access
- Recursive subfolder scanning with configurable depth

### Asset Preview
- **Grid View**: Visual thumbnail preview for quick asset identification
- **List View**: Detailed information including file name, type, size, and path
- **Real Thumbnails**: Actual image previews for textures and 3D mesh renders
- **Type Indicators**: Color-coded badges for quick asset type recognition

### Drag & Drop Import
- Drag assets directly from the Fast Assets window to:
  - **Level Viewport**: Automatically imports and places the asset at cursor location
  - **Content Browser**: Imports to the selected folder
- Multi-select support for batch operations
- Visual feedback during drag operations

### Smart Organization
- Automatic folder organization by asset type:
  - 3D Models → `/Assets/Meshes/`
  - Textures → `/Assets/Textures/`
  - Audio → `/Assets/Sounds/`
  - And more...
- Customizable folder mappings
- Configurable base import path

### Supported File Types

| Category | Formats |
|----------|---------|
| **3D Models** | FBX, OBJ, GLTF, GLB |
| **Textures** | PNG, JPG, JPEG, TGA, BMP, EXR, HDR, PSD |
| **Audio** | WAV, MP3, OGG, FLAC |
| **Unreal Assets** | UASSET, UMAP |
| **Data** | JSON, CSV |

### Settings & Customization
- Configurable thumbnail size (Small/Medium/Large)
- Default view mode preference
- Duplicate handling options (Ask/Skip/Replace/Rename)
- Auto-import toggle
- Performance mode (disable thumbnails for faster browsing)

## Perfect For
- **Level Designers** - Quickly place downloaded assets into your levels
- **3D Artists** - Preview and import models without leaving the editor
- **Audio Designers** - Browse and import sound files efficiently
- **Indie Developers** - Streamline your solo workflow
- **Studios** - Speed up asset integration across teams

## Technical Details
- **Module Type**: Editor Only
- **Platforms**: Windows, Mac, Linux
- **Engine Versions**: 5.4, 5.5, 5.6, 5.7
- **Network Replicated**: No
- **Documentation**: Included

## Installation
1. Copy the plugin to your project's `Plugins` folder
2. Restart Unreal Engine
3. Enable "Fast Assets" in Edit → Plugins
4. Access via Window → Fast Assets

# Fast Assets

**Fast Assets ile asset iş akışınızı hızlandırın - Unreal Engine için en gelişmiş harici asset tarayıcısı.**

Fast Assets, herhangi bir harici klasördeki asset'leri Unreal Engine içinden taramanıza, önizlemenize ve projenize import etmenize olanak tanıyan güçlü bir editor plugin'idir. Sıkıcı dosya gezintisine ve manuel import işlemlerine elveda deyin!

## Temel Özellikler

### Harici Klasör Tarama
- Bilgisayarınızdaki herhangi bir klasörü doğrudan Unreal Engine içinden tarayın
- Tanıdık bir arayüzle dizinler arasında gezinin
- Hızlı erişim için son kullanılan yolları hatırlayın
- Ayarlanabilir derinlikte alt klasör tarama

### Asset Önizleme
- **Grid Görünümü**: Hızlı asset tanıma için görsel thumbnail önizleme
- **Liste Görünümü**: Dosya adı, tip, boyut ve yol dahil detaylı bilgi
- **Gerçek Thumbnail'ler**: Texture'lar için gerçek görüntü önizlemeleri ve 3D mesh render'ları
- **Tip Göstergeleri**: Hızlı asset tipi tanıma için renkli rozetler

### Sürükle-Bırak Import
- Asset'leri Fast Assets penceresinden doğrudan sürükleyin:
  - **Level Viewport**: Otomatik import ve imleç konumuna yerleştirme
  - **Content Browser**: Seçili klasöre import
- Toplu işlemler için çoklu seçim desteği
- Sürükleme sırasında görsel geri bildirim

### Akıllı Organizasyon
- Asset tipine göre otomatik klasör organizasyonu:
  - 3D Modeller → `/Assets/Meshes/`
  - Texture'lar → `/Assets/Textures/`
  - Ses → `/Assets/Sounds/`
  - Ve daha fazlası...
- Özelleştirilebilir klasör eşlemeleri
- Ayarlanabilir temel import yolu

### Desteklenen Dosya Tipleri

| Kategori | Formatlar |
|----------|-----------|
| **3D Modeller** | FBX, OBJ, GLTF, GLB |
| **Texture'lar** | PNG, JPG, JPEG, TGA, BMP, EXR, HDR, PSD |
| **Ses** | WAV, MP3, OGG, FLAC |
| **Unreal Asset'ler** | UASSET, UMAP |
| **Data** | JSON, CSV |

### Ayarlar ve Özelleştirme
- Ayarlanabilir thumbnail boyutu (Küçük/Orta/Büyük)
- Varsayılan görünüm modu tercihi
- Duplikat işleme seçenekleri (Sor/Atla/Değiştir/Yeniden Adlandır)
- Otomatik import açma/kapama
- Performans modu (daha hızlı tarama için thumbnail'leri devre dışı bırakma)

## Kurulum Talimatları

```
1. Plugin ZIP dosyasını indirin
2. ZIP'i açın ve FastAssets klasörünü projenizin Plugins klasörüne kopyalayın:
   YourProject/
   └── Plugins/
       └── FastAssets/
3. Unreal Engine'i yeniden başlatın
4. Edit → Plugins menüsünden "Fast Assets" plugin'ini etkinleştirin
5. Window → Fast Assets menüsünden pencereyi açın
```
